# Splendor Game 

## 1. Description 
This is a Java implemented version of the board game Splendor. 
Players act as Renaissance gem merchants, collecting gems to purchase development cards and accumulate Prestige Points to win the game.

## 2. Features 
- Supports 2–4 players with turn-based gameplay
- Supports Human vs Human, Human vs AI or a mix of both
- Has a scoring system based on the prestige points for victory tracking

## 3. Game Rules (Simplified)

Players take turns performing one of the following actions:
- Take 3 different gem tokens OR 2 tokens of the same type
- Purchase a development card using gems
- Reserve a card

Cards provide:
- Permanent gem bonuses
- Prestige Points which are earned through development cards and nobles

The first player to reach 15 Prestige Points triggers the end of the game.
The player with the highest points wins.

## 4. Installation instructions
1. Clone the repository: git clone https://github.com/ze-2/splendid-splendor.git
2. Navigate to the project directory: cd splendid-splendor
3. Compile the project: ./compile.sh
4. Run the game: ./run.sh
